# code-verify Skill

트리거 키워드: "빌드해줘", "테스트 돌려줘", "lint 확인해줘", "build", "test", "검증해줘"

## 실행 흐름

1. `code-verifier` 에이전트 실행 (실패 시 내부 최대 3회 재시도)
2. 결과 출력 (PASS/FAIL 테이블 + 수정 내역 + 미해결 사항)

## 결과 처리

| 결과 | 처리 |
|------|------|
| 전체 PASS | 완료 |
| FAIL 자동 수정됨 | 재검증 확인 |
| FAIL 수동 필요 | 사용자에게 원인 보고 |
